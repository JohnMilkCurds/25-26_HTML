Tables

The Tabletag opens a table. Each row is defined by a tr tag. Each column within a row is defined by td tag.

There are lots of attributes that tables can use. Some examples are

  • Border - This sets a border for the table tag
  • Width - Sets the width of a table/column
  • Height - Sets the height of a table/column
